# Antonio's Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, work experience, education, and projects. It is designed to be fully responsive and mobile-friendly.

## 🌟 Features

- **Home Section**: Introduction with a typewriting effect using [TypeIt](https://typeitjs.com/).
- **About Section**: Personal profile, skills, and downloadable CV.
- **Resume Section**: Work experience and education, visually structured.
- **Projects Section**: Portfolio of completed and upcoming projects.
- **Contact Section**: Connect via LinkedIn or email.
- **Smooth Navigation**: Internal links and scroll buttons for easy navigation.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## 💻 Technologies Used

- **HTML5** – Structure of the website
- **CSS3** – Styling and responsive layout
- **JavaScript** – Interactive effects and navigation
- **TypeIt.js** – Typewriting animation for the intro
- **SVG icons** – Custom scalable vector icons
